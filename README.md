# AI English Interviewer

Production-ready skeleton for a split-screen English interview app with Realtime LLM adapters (OpenAI/Gemini) and a mock provider.

## Quick start

1. Install deps:
```
npm i
```
2. Create `.env` with:
```
VITE_REALTIME_PROVIDER=openai
VITE_OPENAI_API_KEY=your_openai_api_key
VITE_MODEL=realtime-preview
VITE_N8N_WEBHOOK={{N8N_WEBHOOK_URL}}
```
For mock-only demo:
```
VITE_REALTIME_PROVIDER=mock
VITE_N8N_WEBHOOK=http://localhost:5678/webhook
```
3. Run dev server:
```
npm run dev
```

## Providers
- Set `VITE_REALTIME_PROVIDER` to `openai` or `gemini` to enable chosen provider; `mock` for local demo.

## Testing
```
npm test
```
